Introdução ao HTML na Prática

Apresentando o Curso: Basicamente, nesse primeiro momento o professor "Diego Medeiros" apresentou o percurso até o final da "Introdução ao HTML na Prática" e também citou o objetivo geral. O qual, é aprender a instalar e configurar ambiente de desenvolvimento. Entender o funcionamento do Inspetor de elementos do navegador e criar sua página HTML

Ferramentas Utilizadas: O Visual Studio Code (VS Code) é um editor de código fonte leve e altamente customizável, amplamente utilizado para desenvolvimento web, incluindo HTML. Assim, durante a "Formação HTML Web Developer" será nossa IDE. O mesmo oferece recursos como destaque de sintaxe, completamento automático, depuração integrada e integração com Git, proporcionando um ambiente eficiente e produtivo para escrever e editar código HTML.

Inspetor de Elementos: O Inspetor de Elementos dos navegadores é uma ferramenta de desenvolvimento integrada que permite visualizar e inspecionar o código HTML, CSS e JavaScript de uma página da web em tempo real. Ele oferece recursos como visualização da estrutura do DOM, edição ao vivo de estilos CSS, depuração de JavaScript e análise de desempenho, ajudando os desenvolvedores a entenderem e depurarem suas páginas da web de forma eficiente.

Estrutura Básica do HTML: A estrutura básica do HTML consiste em elementos essenciais para criar uma página web. Inclui o doctype, que define a versão do HTML, as tags "html" para envolver todo o conteúdo, "head" para informações do documento, como metadados e links para estilos e scripts, e "body" para o conteúdo visível na página. Essa estrutura fornece o esqueleto básico para construir uma página web e é o ponto de partida para adicionar conteúdo, estilos e funcionalidades.

Falando sobre Tags e seus Atributos: As tags HTML são elementos fundamentais para estruturar e formatar o conteúdo de uma página web. Elas são cercadas por colchetes angulares e podem ter atributos para definir características adicionais. Dentre elas temos:

  • "h1" a "h6": Para títulos de diferentes níveis de importância.
  • "p": Para parágrafos de texto.
  • "a": Para criar links para outras páginas ou recursos.
  • "ul" e "ol": Para listas não ordenadas e ordenadas, respectivamente.
  • "li": Para itens de lista.
  • "img": Para inserir imagens na página.

Textos: Além das tags citadas acima, é possível estilizar o texto usando atributos como styles e classes, e é possível formatar o texto usando tags como "br" para quebras de linha e "hr" para linhas horizontais. Essas tags ajudam a estruturar e apresentar o conteúdo textual de maneira organizada e legível em uma página web.

Listas Ordenadas e Não Ordenadas: São elementos HTML que permitem organizar e exibir informações de maneira estruturada em uma página web. Listas Não Ordenadas, são listas que não seguem uma ordem específica. Cada item é marcado com um ponto, um círculo ou um quadrado. São definidas usando a tag "ul" e cada item da lista é definido com a tag "li". Listas Ordenadas, são listas que seguem uma ordem numérica ou alfabética. Cada item é automaticamente numerado. São definidas usando a tag "ol" e cada item da lista é definido com a tag "li". Ambas as listas são úteis para organizar itens relacionados e são amplamente utilizadas em conteúdo web para melhorar a legibilidade e a estrutura do texto.

Links: Links no HTML são elementos usados para criar conexões entre diferentes páginas da web ou recursos online. Eles são definidos pela tag "a" e podem ser usados para linkar para páginas externas". Utilizando o atributo "href", você pode especificar o endereço URL de destino. Como também linkar para páginas internas. Os links são uma parte fundamental da experiência de navegação na web, permitindo aos usuários acessar e interagir com diferentes recursos e conteúdos online.
