Historia da Web

Apresentação do Curso e Professor: Basicamente nesse primeiro momento o professor "Diego Medeiros Mainardes", o qual é Desenvolverdor Sênior e trabalha com Desenvolvimento Web a muito tempo apresenta o curso da "Formação HTML Web Developer" e fala sobre os "Primeiros Passos com HTML".

Primeiros Computadores: Os primeiros computadores surgiram no século XX, marcando o início da era da computação. Inicialmente grandes e complexos, como o ENIAC e o Colossus, desenvolvidos durante a Segunda Guerra Mundial, essas máquinas pioneiras foram essenciais para cálculos científicos e atividades militares. Posteriormente, evoluíram para modelos mais compactos e acessíveis, dando origem à revolução da computação pessoal.

Como Surgiu: A Internet surgiu a partir de projetos de pesquisa financiados pelo governo dos Estados Unidos na década de 1960, com o objetivo de criar uma rede de comunicação robusta e descentralizada. O desenvolvimento da ARPANET foi um marco importante, conectando inicialmente computadores de instituições acadêmicas e militares. Com o tempo, a Internet se expande globalmente, conectando bilhões de dispositivos e transformando-se na infraestrutura fundamental da comunicação moderna.

Grandes Nomes: Durante a criação da internet e o desenvolvimento dos primeiros computadores, vários nomes se destacaram por suas contribuições importantes. Entre eles, podemos citar:

  • "Alan Turing", o qual é considerado um dos pioneiros da computação, Turing desempenhou um papel fundamental na quebra do código Enigma durante a Segunda Guerra Mundial e contribuiu com ideias fundamentais para a teoria da computação.
  • "Tim Berners-Lee", o criador da World Wide Web (WWW), Berners-Lee desenvolveu o primeiro navegador web e o protocolo HTTP, permitindo a distribuição global de informações na internet.
  • "Grace Hopper", a qual foi uma das primeiras programadoras de computador, Hopper foi pioneira no desenvolvimento da linguagem de programação COBOL e desempenhou um papel importante no desenvolvimento do primeiro compilador.
  • "Vint Cerf e Bob Kahn", os quais são conhecidos como os "pais da internet", Cerf e Kahn desenvolveram o TCP/IP, protocolo fundamental para a comunicação na rede, possibilitando a interconexão de diferentes redes de computadores, dando origem à internet moderna.
  • Entre outros, que foram nomes fundamentais para pavimentar o caminho para a revolução digital que moldou o mundo contemporâneo.

O que são Clients

Cliente no Linguajar Web: Clientes no contexto da linguagem web referem-se aos navegadores de internet, que são softwares utilizados pelos usuários para acessar e interagir com páginas web. Esses clientes interpretam e exibem conteúdo HTML, CSS e JavaScript, permitindo que os usuários visualizem e interajam com diversos tipos de mídia, como textos, imagens, vídeos e aplicativos web. Exemplos comuns de navegadores incluem Google Chrome, Mozilla Firefox, Microsoft Edge e Safari.

Navegadores: Como dito acima, os navegadores são softwares utilizados para acessar e interagir com conteúdo na internet. Eles interpretam linguagens como HTML, CSS e JavaScript, permitindo aos usuários visualizar páginas da web, vídeos, imagens e outros tipos de mídia. Exemplos populares incluem Google Chrome, Mozilla Firefox, Microsoft Edge e Safari. Os navegadores desempenham um papel fundamental na experiência de navegação na internet, oferecendo recursos como histórico de navegação, abas múltiplas e plugins personalizados.

Aplicações Web: Aplicações web são programas acessados ​​através de um navegador da web, sem necessidade de instalação local. Eles utilizam tecnologias como HTML, CSS e JavaScript para fornecer funcionalidades interativas e dinâmicas aos usuários. Exemplos comuns incluem redes sociais, e-mails, serviços de armazenamento na nuvem e aplicativos de comércio eletrônico. Essas aplicações permitem aos usuários realizar diversas tarefas e acessar informações em qualquer dispositivo com conexão à internet.

Dispositivos Móveis: Dispositivos móveis são dispositivos portáteis, como smartphones e tablets, que oferecem acesso à internet e uma variedade de aplicativos. Eles são amplamente utilizados devido à sua conveniência e mobilidade, permitindo que os usuários acessem informações, realizem transações e se comuniquem em qualquer lugar. Cerca de 99% dos usuários de internet utilizam dispositivos móveis para acessar conteúdo online, tornando-os uma parte essencial da experiência digital moderna. Isso ressalta a importância crescente dos dispositivos móveis no cenário da tecnologia e da comunicação.

O que são Servers

Sobre Servidores: Servidores no contexto da linguagem web são computadores ou softwares especializados que armazenam e disponibilizam informações, páginas da web e outros recursos para os clientes, como navegadores, que os solicitam. Eles são fundamentais para hospedar sites, aplicativos web e serviços online, garantindo que os usuários possam acessar esses recursos a qualquer momento, geralmente através da internet. Os servidores web executam funções como processamento de transporte, armazenamento de dados e entrega de conteúdo aos clientes de forma eficiente e segura.

Hardwares e Softwares em Servidores: Servidores são compostos por hardware e software essenciais para sua operação. Os componentes de hardware incluem exercícios como memória RAM, armazenamento e placas de rede. O software engloba o sistema operacional do servidor, aplicativos e serviços necessários para fornecer funcionalidades específicas, como servidores web, bancos de dados e softwares de segurança. Juntos, hardware e software garantem o funcionamento eficiente e confiável dos servidores.

Tipos de servidores: Existem vários tipos de servidores, cada um projetado para definir funções específicas dentro de uma rede de computadores. Alguns dos principais tipos incluem:

  • Servidores Web: Responsáveis por hospedar e fornecer páginas da web e outros conteúdos para os clientes que acessam a internet.
  • Servidores de Arquivos: Armazenam e agrupam arquivos e recursos de dados entre os dispositivos conectados à rede.
  • Servidores de Banco de Dados: Responsáveis por armazenar, gerenciar e processar dados, permitindo acesso rápido e seguro às informações.
  • Servidores de Email: Gerenciam o envio, coleta e armazenamento de mensagens de correio eletrônico.
  • Servidores de Aplicativos: Hospedam e fornecem acesso a aplicativos específicos, permitindo que os usuários executem tarefas específicas de software.
  • Servidores de Jogos: Oferecem infraestrutura para jogos online, permitindo a conexão de jogadores e o processamento de partidas em tempo real.

Hospedagem de Sites: A hospedagem de sites refere-se ao serviço que permite que indivíduos e empresas acessem seus sites acessíveis na internet. Os provedores de hospedagem oferecem espaço em servidores web para armazenar arquivos, bancos de dados e outros recursos necessários para manter um site online. Além disso, eles fornecem recursos como largura de banda, suporte técnico e segurança para garantir o funcionamento estável e seguro dos sites hospedados.

Linguagens de Programação

O que é: Linguagens de programação são conjuntos de regras e instruções que permitem aos desenvolvedores escreverem códigos para criar software e aplicativos. Eles são usados ​​para definir a lógica e o comportamento dos programas, permitindo que os computadores executem tarefas específicas. Existem diversas linguagens de programação, cada uma com suas características e específicas diferentes, mas todas abordadas com o objetivo de facilitar a comunicação entre humanos e computadores.

Server-Side: Referem-se a parte de um aplicativo web executado no servidor. Nesta abordagem, o processamento e a lógica de negócios são realizados no servidor antes de enviar o resultado para o cliente. Isso inclui manipulação de dados, interações com bancos de dados, autenticação de usuários e outras operações que ocorrem no back-end da aplicação. O Server-Side é essencial para garantir segurança, eficiência e controle centralizado em sistemas web.

Client-Side: Referem-se a parte de uma aplicação web que é realizada no navegador do cliente, ou seja, no dispositivo do usuário. Nesta abordagem, o processamento e a interação com a interface ocorrem no próprio dispositivo do usuário, utilizando tecnologias como HTML, CSS e JavaScript. Isso permite uma experiência interativa e responsiva para o usuário, com manipulação de dados e elementos da página ocorrendo localmente, sem a necessidade de comunicação frequente com o servidor. O Client-Side é fundamental para criar interfaces dinâmicas e interativas em aplicações web.

HTML não é Linguagem de Programação: HTML, abreviatura de HyperText Markup Language, é uma linguagem de marcação usada para criar a estrutura e o conteúdo de páginas web. Embora seja essencial para o desenvolvimento web, não é considerada uma linguagem de programação no sentido tradicional. Ao mesmo tempo, o HTML é uma linguagem de marcação que define a estrutura e a semântica dos elementos em uma página, permitindo a organização e apresentação de conteúdo na web. Ele não possui recursos de lógica ou processamento dinâmico de dados, como ocorre em linguagens de programação, como JavaScript ou Python.
